Vue uses a representation of the actual DOM, because DOM-Updates are not very performant. Vue creates a Virtual-DOM.

The VueJS-Instance Lifecycle

The VueJS Instance Lifecycle starts with the new Vue() constructor. It uses the beforeCreate() method to initialize Data & Events, executing created() when finished. Then the template will be compiled and we call beforeMount(). At this point the compiled html will mount to the actual DOM. Data changed => beforeUpdate(), Desctruction of the Vue Instance: beforeDestroy(), afterwards destroyed().
